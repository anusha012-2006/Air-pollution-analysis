# Air Pollution Prediction and Analysis

## 📌 Project Overview
This project focuses on analyzing air pollution data and predicting pollution levels using machine learning techniques.

## 🎯 Objectives
- Analyze pollution data using EDA
- Detect patterns and outliers
- Build a prediction model using Linear Regression

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- Scikit-learn

## 📊 Key Features
- Data Cleaning and Preprocessing
- Visualization (Histogram, Boxplot, Heatmap)
- Air Quality Classification
- Linear Regression Model
- Model Evaluation (R², MSE, RMSE)

## 📈 Results
- Strong correlation between pollutant_min, pollutant_max, and pollutant_avg
- Achieved high accuracy (R² ≈ 0.91)

## 🚀 Future Scope
- Use advanced models like Random Forest
- Add weather data
- Build real-time dashboard
